------------------------------------------------------------
Installation
------------------------------------------------------------
1. PHP5, MySQL und Apache besorgen. (Am einfachsten fuer 
Windows/Linux/Solaris:  
http://www.apachefriends.org/de/xampp.html )

2 Smarty installieren (http://smarty.php.net/).

3. source/config.inc.php5 editieren

4. Datenbank einrichten (z.B. die MySQL-Kommandos in 
database.sql ausfuehren)

5. source/ auf den server packen. (Wie in config.inc.php5
eingerichtet)

6. Viel Spass
